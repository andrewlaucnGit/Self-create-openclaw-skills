A safe shared-pool and workspace cleanup skill built around a **quarantine-first** workflow.

This skill is designed for identifying and safely handling stale:
- `.bak`
- `.disable`
- deprecated backups
- disabled directories
- leftover recovery artifacts

Includes:
- inspection guidance
- quarantine and staged deletion workflow
- rollback-aware validation
- cleanup scripts and runbook

Use this skill when you want to reduce operational clutter without risking deletion of live files.
